# PWA Icons

Place your PWA icon image here:

- **File name:** `icon.png`
- **Recommended size:** 512×512 px (square PNG; browsers will scale for smaller sizes)
- **Format:** PNG (with or without transparency)

The app uses this file for:
- Web App Manifest (Add to Home Screen)
- Apple Touch Icon
- Push notification icon

If your file has a different name, rename it to `icon.png`.
